# Fighting The Undead - 3D Fantasy Game Enginewith THREE.js 
Complete development of an exploratory 3D video game with real-time architecture, including:
- 3D rendering and scene management using Three.js
- Procedural generation of the map and game objects
- Implementation of gameplay logic, enemy AI, and interaction system
- Collision management, animations, and user input
- Integration of 3D models and dynamic lighting
